Settings

Your API key is stored locally on this device and never shared.

How to get your API key

  1. Open clockify.me and sign in to your account.
  2. Click your avatar in the top-right corner and choose Profile Settings.
  3. Scroll to the bottom of the page to the API section.
  4. Click Generate (or copy the existing key) and paste it into the field above.

What data is accessed

The app only reads your own time entries using the standard Clockify personal API key — no special permissions or admin access are required. Specifically, it fetches:

  • Your user profile (to identify your workspace)
  • Your time entries for the selected year

No data is written back to Clockify and nothing leaves this device.
